# taskflow

一个的任务清单软件

## 当前功能

- 底部导航：任务、专注、日历、倒数日
- 任务侧边栏分类：今天、明天、最近7天、收集箱、已经完成、垃圾桶、已经放弃
- 任务支持本地记忆（持久化）
- 点击任务进入详情编辑：时间、名字、描述、所在位置

## 快速运行

```powershell
flutter pub get
flutter run
```

## 运行测试

```powershell
flutter test
```
